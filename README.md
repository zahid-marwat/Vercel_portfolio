## ğŸš€ Quick Start

1. **Clone the repository**:
   ```bash
   git clone https://github.com/zahid-marwat/Vercel_portfolio.git
   cd Vercel_portfolio
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Start development server**:
   ```bash
   npm run dev
   ```

4. **Open in browser**:
   Open [http://localhost:3000](http://localhost:3000) to view the portfolio.
